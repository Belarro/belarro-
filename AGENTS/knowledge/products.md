## How to Add a New Product

When Ron wants to add a new variety:

1. **Grower** confirms: grow time, yield per tray, seed cost, light days
2. **Bookkeeper** calculates: COGS/tray, margin at current pricing
3. **Builder** adds: to Supabase products table + website
4. **Storyteller** creates: variety spotlight content
5. **Update this file:** Move from INACTIVE to ACTIVE, or add new entry

Required data for any new product:
- Product name (EN + DE)
- Category (Sprossen / Mikrogreens / Feinkräuter / Mischungen)
- Yield (grams per tray)
- Grow days (seed to harvest)
- Seed cost per kg (incl. shipping)
- Seeds per tray (grams)
- Light days (how many days under lights)
- Container size (750ml or 2000ml)

## Key Product Facts
- **Growing method:** Soil-grown (not hydroponic)
- **Soil:** 3L per tray, 0.50 EUR/tray
- **Harvest:** Same-day as delivery
- **Shelf life:** 7-10 days refrigerated
- **Delivery:** Tuesdays only, free, no minimum order
- **Container strategy:** 750ml (0.145 EUR) and 2000ml (0.253 EUR) only
